如何修改内容

  1. 解压此压缩包到一个目录中。
  2. 改动文字性内容：用代码编辑软件打开 index.html 文件，参考其中的注释进行修改。
  3. 改动图片附件：在 attachments 目录中，找到需要更改的图片，替换它。
  4. 更多改动，请访问 h5.saiwon.com ，下载软件或进入网页版后，在开始界面中的输入框中粘贴以下内容：
